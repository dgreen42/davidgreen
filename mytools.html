The main languages that I write in are R and Rust. I use R for data analysis and model building. R has good support for GIS, Baysian statistics (through Stan which actaully uses a C interface), and has a large ecosystem of packages that makes life easier when it comes to biological studies. R is also highly vectorized making it easy to work with data structures common in many research fields. Rust is compiled and fast. It is my go to for making CLI tools that I use when working with massive data sets. I could learn C but... segfaults.
